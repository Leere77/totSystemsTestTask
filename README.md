## Тестовое задание для TOT Systems
Запуск
- Загрузить репозиторий
- В директории репозитория `npm install`
- `npm start` для запуска dev-server
- `npm run-script build` для production сборки 

Стек
- react
- redux
- react-router
- sass

Бэкенд (работает на heroku)
- express.js
- mongoose
